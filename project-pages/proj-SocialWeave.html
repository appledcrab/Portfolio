My Project Experiences

This was the second group project for my Mobile App Development Class. I worked with the same person I did before on the Mood Mate app.

To speed up the process I looked for more packages that could help. I found a peristent nav bar package and a basic messaging that helped with messaging between users.

It was a little more stressful doing this alongside another large group project, but we pushed through. Once it was starting to work, having messages in real time was fun and I tested it with my friend and my phones.

Lessons Learned

Importance of reading through not only the documentation, but also following the methods locally to see exactly what it's doing if the documentation isn't helping.

I learned more about the firebase database along with having collections within collections and having to work around them aswell.

Creating unique user experiences by having their page shown by the list of people they are following.
